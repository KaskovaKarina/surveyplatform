📤 Загрузка файлов опросов

Поддерживаются форматы: Excel (.xlsx, .xls) и CSV

📁
Нажмите для выбора файлов
или перетащите файлы сюда
💡 Важно! Для корректной работы пайплайна каждый файл опроса должен содержать колонку с идентификатором студента (ID). После загрузки файлов перейдите в раздел "Мои файлы", чтобы настроить соответствие колонок.

🚀 Как это работает

1️⃣

Загрузка

Загрузите файлы опросов из любых источников (Яндекс.Формы, Google Forms)

2️⃣

Настройка

Укажите, в какой колонке каждого файла находится ID студента

3️⃣

Обработка

Пайплайн автоматически объединяет все опросы в единую таблицу

4️⃣

Анализ

Скачайте результат или проводите анализ прямо на платформе
