📌 Важно! Анкета студентов (справочник)

Для корректной работы пайплайна создайте отдельный опрос "Анкета студента", где студенты укажут свои данные:

  • СНИЛС (обязательно, будет использоваться как идентификатор)
  • Группа
  • Стипендия
  • Факультет

Как назвать файл:

  • Назовите файл с анкетой так, чтобы в названии было слово "reference" или "анкета"
  • Например: reference_students.xlsx или анкета_студентов.xlsx
  • Система автоматически распознает этот файл как справочник

💡 Совет: Попросите студентов заполнить анкету один раз в начале учебного года. Затем используйте эту выгрузку для всех последующих опросов.

📖 Инструкция по работе с СНИЛС

  1. Шаг 1: Создайте и загрузите анкету студентов (файл с СНИЛС, ФИО, группой, стипендией, факультетом)
  2. Шаг 2: Создайте опросы (столовая, аудитории, преподавание и т.д.) с обязательным полем "ФИО"
  3. Шаг 3: Загрузите все файлы опросов на платформу
  4. Шаг 4: Для каждого файла опроса выберите колонку с СНИЛС
  5. Шаг 5: Нажмите "Запустить пайплайн"
  6. Шаг 6: Система объединит все опросы по СНИЛС и добавит данные из анкеты
⚠️ Важно: Студент должен везде указывать СНИЛС одинаково.
